# **e-Ponido 仕様書 (e-Ponido 言語仕様)**

---

## **Baso-1: 基本概念**

### **1.1 e-Ponidoの定義**
- **Ponidoの拡張**：
  - Ponidoの簡潔な語彙と文法を利用。  
  - 哲学的な簡潔性と国際的な互換性を重視。  
- **AI技術の統合**：
  - **RAG**（情報検索強化）：テキストの検索と拡張。  
  - **MCP**（複数条件処理）：複雑な条件と技術処理に対応。  
- **適応性と拡張性**：
  - コミュニティ主導で開発。  
  - VRやARとの連携を視野に入れる。  

### **1.2 e-Ponidoの目的**
- **コミュニティ支援**：多国間の利用をサポート。  
- **技術対応**：VRやAI技術の使用を促進。  
- **国際化**：幅広い利用者に理解されることを目指す。

## **Baso-2: 音と文字システム**

### **2.1 音の構成**
- **母音**：a, i, u, e, o  
- **子音**：k, s, t, n, h, m, r, y, w  
- **音のルール**：  
  - CV（子音＋母音）の音節構造を採用。  

### **2.2 文字システム**
- **ローマ字表記**：1音＝1文字の規則。  
- **Ponido文字**：将来的に文化的な文字システムの導入を検討。  

---

## **Baso-3: 文法規則**

### **3.1 語順**
- **SOV構造**（主語－目的語－動詞）：  
  ```plaintext
  Mi-ga numero-o studi-desu.   # 私は数字を学ぶ。  
  Ana-ga lingvo-o krea-desu.   # あなたは言語を作る。  
  ```

### **3.2 助詞システム**
- **-ga**：主語を示す。  
- **-o**：目的語を示す。  
- **-wa**：主題を示す。  
- **-ni**：場所を示す。  
- **-e**：方向を示す。  

### **3.3 時制システム**
- 動詞は「-u」で終わる。  
- **時制の表現**：  
  - 過去：*tempo-pase*  
  - 現在：*tempo-now*  
  - 未来：*tempo-future*  

### **3.4 接辞ルール**
- **名詞形**：-mo  
- **形容詞形**：-i  
- **副詞形**：-li  
- **命令形**：-ka

## **Baso-4: 語彙と拡張**

### **4.1 基本語彙**
- **基礎語彙数**：50～200語。  
- **国際的な語彙**：  
  - tempo（時間）, futuro（未来）, spaco（空間）, mov（動く）, krea（創造）。  

### **4.2 語彙の評価基準**
1. その語彙は必要か？  
2. 簡潔で学びやすいか？  
3. 国際的に理解されるか？  
4. 多用途に使用できるか？

---

## **Baso-5: 数詞システム**

### **5.1 基本数詞**
- **基礎数詞**：  
  ```plaintext
  1: un  
  2: du  
  3: tri  
  4: kvar  
  5: kin  
  ```

- **十進法**：  
  - 20 → *du-deka*  
  - 21 → *du-deka-un*  
  - 1,234 → *un-mil-du-cent-tri-deka-kvar*  

### **5.2 計算規則**
- **計算例**：  
  ```plaintext
  Plus: un-plus-du-wa tri-desu.      # 1 + 2 = 3  
  Minus: tri-minus-un-wa du-desu.   # 3 - 1 = 2  
  Multi: du-multi-du-wa kvar-desu.  # 2 × 2 = 4  
  Divid: kvar-divid-du-wa du-desu.  # 4 ÷ 2 = 2  
  ```

## **Baso-7: 技術統合**

### **7.1 RAG（情報検索強化）**
- **例文**：  
  ```plaintext
  Sistema-ga teksto-o analizu-desu.        # システムがテキストを分析する。  
  ```

### **7.2 MCP（複数条件処理）**
- **例文**：  
  ```plaintext
  Joken-1-ga hontou no toki:               # 条件1が真の場合：  
    Action-1-o okonau.                    # アクション1を実行する。  
  ```
